# 第六週 - Vue Cli、Vue Router

## 主線任務

- 使用 Vue Cli 完成路由設定
- 在 Vue Cli 中完成登入、前台表單驗證功能
- 建立 .env 檔案內容

```
VUE_APP_API=
VUE_APP_PATH=
```

作業須符合此[作業規範](https://hackmd.io/XbKPYiE9Ru6G0sAfB5PBJw)
